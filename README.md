 # 📊 NumPy & Pandas Coding Assignment

## 📌 Overview

This project demonstrates fundamental data analysis skills using **NumPy** and **Pandas**.
It covers real-world tasks such as salary analysis, sales trends, data cleaning, and time series analysis.

The objective is to apply **vectorized operations**, write clean code, and extract meaningful insights from data.

---

## 🛠️ Tools & Libraries

* Python
* NumPy
* Pandas
---

## 📂 Project Structure

### 🔹 Q1. Employee Salary Adjustment

* Increased salaries by 10%
* Calculated average, maximum, and minimum salary
* Filtered employees earning above 60,000

---

### 🔹 Q2. Sales Data Analysis

* Computed total sales
* Calculated mean and standard deviation
* Identified high-performing days
* Applied Min-Max normalization

---

### 🔹 Q3. Matrix Operations for Business Data

* Calculated total sales per product and region
* Identified best-performing product
* Found region with highest sales

---

### 🔹 Q4. Employee Dataset Exploration

* Created a DataFrame from dictionary
* Generated summary statistics
* Analyzed salary by department
* Filtered experienced employees
* Added bonus column

---

### 🔹 Q5. Missing Data Handling

* Detected missing values
* Imputed missing data using mean and median
* Cleaned dataset by removing remaining null values

---

### 🔹 Q6. Sales Performance Analysis

* Aggregated sales by region
* Identified top salesperson
* Computed average sales
* Sorted dataset by performance

---

### 🔹 Q7. Customer Purchase Analysis

* Analyzed average purchase by city
* Identified high-value customers
* Found top revenue-generating city
* Created age-based customer segments
* Evaluated spending patterns by age group

---

### 🔹 Q8. Time Series Sales Analysis

* Converted date column to datetime index
* Calculated monthly growth rate
* Applied rolling average for trend analysis
* Identified highest growth period

---

## 📈 Key Skills Demonstrated
